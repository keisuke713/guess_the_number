# Guess the number

## usage
- you type two numbers
- we randomly generate number between them
- guess what a number is

## example
```
guess_the_number % go run .
This is the Guess the Number game.
You type min number and max number.
Then, we generate a number between two number you just typed
And you guess what it is!

Type min number: 1
Type min number: 5
Guess the number: 1
Oops!!!
Not the one!!
You can try as much as you want
Guess the number: 3
You did it!!!
3 is the one we selected
```
